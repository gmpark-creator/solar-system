# 태양계 프로젝트

> DDuim(축구 하이라이트 엔진)과는 **완전히 별개**인 독립 프로젝트.
> 별도 폴더 · 별도 git 레포 · 별도 GitHub 레포로 운영한다.

## 개요

실시간 데이터 기반의 **3D 천체 시뮬레이터**.
구글어스를 최대로 축소했을 때 우주에서 지구와 행성들이 보이는 것처럼,
현재 실제 시각에 맞춰 태양·지구·달(추후 전 행성)이 배치·운동한다.

- 기술 스택: **HTML5 + JavaScript + Three.js (CDN)**
- 최종 목표: 달 공전 정밀 계산 → 지구 조석(물때표) 계산으로 확장

## 실행 방법

`index.html` 을 더블클릭해서 브라우저로 열면 된다.
(Three.js를 CDN에서 받아오므로 **최초 실행 시 인터넷 연결 필요**)

> 일부 브라우저에서 화면이 안 뜨면 폴더에서 로컬 서버로 여세요:
> `python -m http.server` 실행 후 `http://localhost:8000` 접속.

### Step 1 구현 내용 (`index.html`)

- 우주 배경(별 9,000개) + OrbitControls(드래그 회전 / 휠 줌)
- 태양(중심) · 지구(실시간 자전·공전) · 달(실시간 공전)
- **실제 현재 시각 기반 천체 위치 계산**
  - Schlyter 궤도력 + 달 섭동항 12개 → 지구–달 상대위치·주기를 실제 시간과 정밀 일치
  - 검증: 지구–태양/달 거리, 항성월 주기 복귀 오차 0.9° 통과
- 지구 자전축 23.4° 기울기(고정) / 달 조석 고정(같은 면이 지구를 향함)
- 태양광 PointLight → 지구·달의 낮밤 경계선 및 달 위상 자연 표현
- [시각화 최적화] ↔ [실제 비율] 스케일 토글
- 시간 배속 컨트롤(역행 ~ 1주/초), 일시정지, "지금으로" 리셋
- HUD: 모의 시각, 자전각, 달 위상·조도, 지구–달 / 지구–태양 거리
